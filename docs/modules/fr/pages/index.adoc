= Documentation SysBot

== Vue d'ensemble

SysBot est une bibliothèque complète d'automatisation de tests système qui fournit une interface unified pour se connecter et tester divers systèmes via différents protocoles. Conçue avec Robot Framework à l'esprit, elle offre un support robuste pour les suites de tests Robot Framework et Python unittest.

=== Fonctionnalités

* *Support multi-protocole* : SSH, HTTP, WinRM, Socket (TCP/UDP)
* *Tunneling SSH* : Support des tunnels SSH imbriqués avec gestion automatique
* *Multiplateforme* : Support des systèmes Linux et Windows
* *Intégration Robot Framework* : Support intégré pour l'automatisation Robot Framework avec portée SUITE
* *Architecture modulaire* : Chargement et découverte dynamiques des composants (modules et plugins)
* *Gestion des connexions* : Cache de session robuste et gestion du cycle de vie
* *Gestion des secrets* : Stockage et récupération sécurisés des données sensibles

== Exemple rapide

[source,python]
----
from sysbot.Sysbot import Sysbot

# Créer une instance SysBot
bot = Sysbot()

# Ouvrir une session SSH
bot.open_session(
    alias="mon_serveur",
    protocol="ssh",
    product="bash",
    host="192.168.1.100",
    port=22,
    login="nom_utilisateur",
    password="mot_de_passe"
)

# Exécuter une commande
result = bot.execute_command("mon_serveur", "ls -la")
print(result)

# Fermer toutes les sessions
bot.close_all_sessions()
----

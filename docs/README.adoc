== Viewing Documentation

=== AsciiDoc Viewers

These AsciiDoc files can be viewed:

* **On GitHub**: GitHub natively renders AsciiDoc files
* **Using AsciiDoctor**: Install AsciiDoctor and convert to HTML
* **Using an IDE**: Many IDEs have AsciiDoc preview plugins (VS Code, IntelliJ, etc.)

=== Converting to HTML

To convert documentation to HTML:

[source,bash]
----
# Install AsciiDoctor
gem install asciidoctor

# Convert a single file
asciidoctor docs/index.adoc

# Convert all files
asciidoctor docs/*.adoc
----

=== Converting to PDF

To convert documentation to PDF:

[source,bash]
----
# Install AsciiDoctor PDF
gem install asciidoctor-pdf

# Convert to PDF
asciidoctor-pdf docs/index.adoc
----
